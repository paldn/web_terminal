paramiko是python进行ssh连接的库

tornado是python 多IO复用服务框架

webssh是项目主目录


paramiko和tornado是webssh主要依赖的模块


config.json里是服务器启动的配置

webserver.py是项目运行入口程序



附：socket连接比较消耗服务器资源，请不要一次性打开太多远程连接终端
